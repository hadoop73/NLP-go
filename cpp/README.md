<h2 id='id2'>二位数组</h2>

[学习C++ -> 二维数组](http://www.cnblogs.com/mr-wid/archive/2012/12/11/2813299.html)

**二维数组定义**
```
数据类型  数组名[常量表达式1][常量表达式2];
int a[2][4];
```

**二维数组初始化**
```
数据类型 数组名[常量表达式1][常量表达式2]={初始化数据};
```

- 分行初始化

```
int a[2][3] = {{1,2,3},{4,5,6}};
```

- 不分行初始化

```
int a[2][3] = {1,2,3,4,5,6};
```

- 部分数据元素初始化
```
int a[2][3] = {{1,2},{4}};
```

- 省略第一维的定义,但不省略第二维的定义
```
int a[][3] = {1,2,3,4,5,6};
```
